# Mathster

![](/img/Banner.png)

**Mathster** is an Open Source calculator, focused on simplicity and ease of use ⚡.

## ⭐ Features

* Simple design
* Multi-line editor
* Highlighting of operations
* Unit calculation and conversion
* And more coming soon... 🤩
